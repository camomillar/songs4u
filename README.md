# songs4u 🎵

A personal project inspired by an old, lovely tradition: making someone a mixtape.

Back then, we would burn songs onto a CD, scribble the tracklist by hand, and give it to someone who mattered. songs4u brings that same nostalgic feeling into the digital world.

Made for Valentine's Day, but honestly, any day is a good day to make someone a playlist. ♥

🔗 Try it: [www.songs4u.online](https://www.songs4u.online)
