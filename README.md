# songs4u 🎵

A personal project born from a simple idea — what if you could make a playlist for someone special, just like we used to do back in the day?

There's something nostalgic about those times when we would burn songs onto a CD, write the tracklist by hand, and give it to someone we cared about. **songs4u** brings that feeling back in a digital way.

You pick the songs, write a little message, and the recipient gets to open a virtual CD case — with the tracklist written on the disc, just like the real thing.

Made for Valentine's Day, but honestly, any day is a good day to make someone a playlist. ♥
